CSCE242-001 - Client-Server Computing

Kelly Thao

A Web Applications course where students will use and learn HTML and CSS to design the front-end of a website. JavaScript for functionality to their website. As well as learning how to use node and server side JavaScript to store and retrieve data from a server.

Assignments

  • Assignment 01 - Basic HTML

    As the first assignment of this class, I got to learn how to properly format HTML code, use title tags, headings, bolds, tables, create order and unorder lists and more!

  • Assignment 02 - Basic CSS

    In this assignment, I learned how to link to a stylesheet, style a div, and explore color resoures provided to me.

  • Assignment 03 - Page Layout

    I continue learning about HTML structural elements and how to use and style flexboxes and media queries.

  • Assignment 04 - Recreate CSS Page

    In this assignment, I recreated a CSS page based on the College of Arts and Sciences page from the Univeristy of South Carolina's webpage.

  • Assignment 05 - Introduction Buttons, Functions, and More

    In this assignment, I used javascript arrow functions, constants, and all the conections used in in-person class.

  • Assignment 06 - Conditionals and If Statements.

    In this assignment, I created 2 fun exercises where I get to type a way of transportation and the image shows up, and then I get to choose the color of the heart by a click of a button.
